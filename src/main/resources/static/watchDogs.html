Requisitos mínimos :

* Sistema operativo: Windows 7 SP1, Windows 8.1 o Windows 10 (solo versiones de 64-bit)

* Procesador: Intel Core i5-2400S @ 2.5 GHz / AMD FX 6120 @ 3.5 GHz

* RAM: 6 GB

* Tarjeta gráfica: Nvidia GeForce GTX 660 (2GB) / AMD Radeon HD 7870 (2GB) o superior

* Disco duro: 50 GB de espacio disponible

* Sonido: Compatible con DirectX con los últimos controladores instalados

* Periféricos: Ratón y teclado compatibles con Windows / Mando Microsoft Xbox One / Mando DualShock 4

Requisitos óptimos :

* Sistema operativo: Windows 7 SP1, Windows 8.1 o Windows 10 (solo versiones de 64-bit)

* Procesador: Intel Core i5-3470 @ 3.2 GHz / AMD FX 8120 @ 3.9 GHz

* RAM: 8 GB

* Tarjeta gráfica: Nvidia GeForce GTX 780 (3GB) / GTX 970 (4GB) / NVIDIA GeForce GTX 1060 (3GB) o superior / AMD Radeon R9 290 (4GB) o superior

* Disco duro: 50 GB de espacio disponible

* Sonido: Compatible con DirectX con los últimos controladores instalados

* Periféricos: Ratón y teclado compatibles con Windows / Mando Microsoft Xbox One / Mando DualShock 4
