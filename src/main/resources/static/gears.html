Requisitos mínimos :

* Requiere un procesador y un sistema operativo de 64 bits

* SO: Windows 7 SP1 o Windows 10 de 64 bits

* Red: Conexión de banda ancha a Internet

* Almacenamiento: 80 GB de espacio disponible

* Tarjeta de sonido: Compatible con DirectX

* Procesador: AMD FX-6100 @ 3.90 GHz o Intel Core i3-6100 @ 3.70 GHz

* Memoria: 8 GB de RAM

* Gráficos: AMD Radeon R9 280 o Nvidia GeForce GTX 760

* DirectX: Versión 12

Requisitos óptimos :

* SO: Windows 10 de 64 bits

* Procesador: AMD Ryzen 3 1300 @ 3.40 GHz | Intel Core i5-6500 @ 3.80 GHz

* Memoria: 8 GB de RAM

* Gráficos: AMD Radeon RX 570 o Nvidia GeForce GTX 970 (Windows 10)
